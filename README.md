# claude-skills

Claude skills I built for my own brain.

This repository is a growing collection of custom skills for Claude — each one solving a real problem I ran into, built around how I actually think and learn.

---

## Skills

| Skill | What it does |
|---|---|
| [adhd-notes](./adhd-notes/) | Concise, scannable conspects from courses, articles, and PDFs — optimized for ADHD review |
| [anti-inflammatory-diet](./anti-inflammatory-diet/) | Personalized anti-inflammatory diet assistant — recipe adaptation, product checks, meal suggestions |
More coming.

---

## How to use

Each skill folder contains:
- `SKILL.md` — the skill source file
- `*.skill` — packaged version ready to install in Claude
- `README.md` — what the skill does and how to trigger it
